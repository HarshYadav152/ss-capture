Installation Guide

Follow The Steps Below To Install The SS-Capture Browser extension. Each Section is specific to a browser and written for beginners.

Install on Chrome/Edge
  1. Download the Latest Release From the GitHub Releases page.
    [Screenshot: GitHub Releases Page]
  2. Open Chrome or Edge and go to: chrome://extensions/
    [Screenshot: Chrome Extensions Page]
  3. Enable Developer mode using the toggle at the top-right corner.

    Developer Mode Allows You to Load Extensions Manually for Development or Testing.

    [Screenshot: Developer Mode Enabled]
  4. Click Load unpacked and select the dist/chrome (or dist/edge) folder.
    [Screenshot: Load Unpacked Folder]
  5. The Extension will now appear in the Extensions List and is Ready to use.
    [Screenshot: Extension Installed Successfully
Install on Firefox
  1. Download the Latest Release from the GitHub Releases page.
    [Screenshot: GitHub Releases Page
  2. Extract the ZIP file to a known location.
    [Screenshot: Extract ZIP File]
  3. Open Firefox and navigate to: about:debugging
    [Screenshot: The Firefox Tab]
  4. Click Load Temporary Add-on and select manifest.json from the dist/firefox folder.
    [Screenshot: Load Temporary Add-on]
  5. The extension is now loaded temporarily and will remain active until Firefox is closed.
    [Screenshot: Firefox Extension Loaded]
Troubleshooting
  • If the extension does not appear, confirm that Developer mode is enabled.
  • Ensure you selected the correct folder: dist/chrome, dist/edge, or dist/firefox
  • Firefox temporary add-ons are removed when the browser is restarted.
  • Chrome internal pages such as chrome:// cannot be captured due to browser security restrictions.
